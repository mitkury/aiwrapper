# Captured Responses API streams

These files are raw Server-Sent Event captures used when developing and debugging the OpenAI Responses stream handler.

- `openai-response-text.txt`: text output.
- `openai-response-tools.txt`: function calling.
- `openai-response-image.txt`: image generation.
- `openai-response-text-reasoning.txt`: reasoning summaries followed by text.

They are protocol fixtures, not user-facing examples. Model identifiers and payload fields reflect the time each stream was captured and should not be treated as current defaults.
